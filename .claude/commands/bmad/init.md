# BMAD Workflow Initialization

Welcome to **BMAD** (Build More, Architect Dreams) for Image Optimizer Pro!

## Project Overview
This project is rebuilding Image Optimizer Pro - a SaaS platform for AI-powered image optimization.

## Available Agents
Use these slash commands to activate specialized agents:

| Command | Agent | Role |
|---------|-------|------|
| `/bmad/analyst` | Ana | Business Analyst - Requirements & Analysis |
| `/bmad/pm` | John | Product Manager - PRD & Prioritization |
| `/bmad/architect` | Alex | Solutions Architect - Technical Design |
| `/bmad/dev` | Devon | Full-Stack Developer - Implementation |
| `/bmad/ux` | Uma | UX Designer - User Experience |
| `/bmad/sm` | Sam | Scrum Master - Sprint Planning |
| `/bmad/qa` | Quinn | QA Engineer - Testing |

## Recommended Workflow

### Phase 1: Planning (BMad Method)
1. `/bmad/analyst` - Create project brief, analyze requirements
2. `/bmad/pm` - Create PRD with prioritized features
3. `/bmad/architect` - Design technical architecture
4. `/bmad/ux` - Create UX specifications

### Phase 2: Implementation
1. `/bmad/sm` - Plan sprints, create stories
2. `/bmad/dev` - Implement features
3. `/bmad/qa` - Write tests, ensure quality

## Quick Start Options

**Option A: Full BMad Method** (Recommended for new projects)
Start with `/bmad/analyst` to create a project brief

**Option B: Quick Flow** (For smaller tasks)
Start with `/bmad/dev` for direct implementation

**Option C: Architecture First** (When design is critical)
Start with `/bmad/architect` to design the system
